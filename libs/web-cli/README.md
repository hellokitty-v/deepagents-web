# Dolphin 公安智能体平台 - 产品原型 Demo

> 基于 Manus 架构设计的公安领域通用智能体平台 MVP 原型

## 项目简介

Dolphin 是面向公安机关的通用智能体平台，支持通过自然语言描述任务，AI Agent 自主完成从信息搜集、文档生成、网站创建到数据可视化分析的全流程。

## MVP 功能范围

本原型 Demo 实现了以下核心功能：

### 1. 三大任务类型

- **制作幻灯片**：生成工作汇报、情报简报、案件汇报等 PPT
- **创建网站**：生成信息公示网站、专题网站、数据展示网站
- **数据可视化**：生成数据分析图表、态势展示、统计报表

### 2. 核心页面

- **首页**：任务输入 + 快捷标签 + 推荐任务样例
- **任务执行页**：实时展示 Agent 执行步骤 + 文件预览
- **库页面**：统一管理所有产出文件
- **所有任务页**：查看历史任务列表

### 3. 公安场景适配

- 预置公安业务推荐任务（治安周报、专项行动、禁毒宣传等）
- 公安主题模板（公安蓝配色）
- 公安业务提词引导

## 技术栈

- **前端**：原生 HTML + CSS + JavaScript
- **UI 图标**：Font Awesome 6.4.0
- **无需构建工具**：可直接在浏览器中运行

## 快速开始

### 方式一：直接打开（推荐）

1. 双击 `index.html` 文件，在浏览器中打开

### 方式二：使用本地服务器

```bash
# 使用 Python 3
python -m http.server 8080

# 或使用 Python 2
python -m SimpleHTTPServer 8080

# 或使用 Node.js (需要先安装 http-server)
npx http-server -p 8080
```

然后在浏览器中访问：`http://localhost:8080`

## 功能演示

### 1. 创建任务

- 在首页输入框中输入任务描述
- 或点击快捷标签（幻灯片/网站/可视化）查看推荐任务
- 点击推荐任务卡片自动填充任务描述
- 点击发送按钮开始任务

### 2. 查看任务执行

- 任务开始后自动跳转到执行页面
- 左侧实时展示 Agent 执行步骤
- 右侧预览正在生成的文件
- 进度条显示当前步骤

### 3. 管理文件

- 点击左侧导航"库"查看所有产出文件
- 支持按文件类型筛选
- 点击文件可下载（Demo 版本为模拟）

### 4. 查看历史任务

- 点击左侧导航"所有任务"查看历史记录
- 点击任务可查看详情和产出文件
